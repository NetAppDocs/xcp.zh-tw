= 記錄傾印
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
NFS `logdump` 命令會根據移轉 ID 或工作 ID 來篩選記錄、並將這些記錄傾印至 `.zip` 目前目錄中的檔案。。 `.zip` 檔案的名稱與命令所使用的移轉或工作 ID 相同。

.語法
[source, cli]
----
xcp logdump -m <migration ID>
xcp logdump -j <job ID>
----
