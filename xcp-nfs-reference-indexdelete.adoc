下表列出 `indexdelete` 參數及其說明。

[cols="2*"]
|===
| 參數 | 說明 


| <<nfs_indexdelete_match,索引刪除 - 符合  ；篩選器 >   | 僅處理與篩選器相符的檔案和目錄。 


| <<nfs_indexdelete_loglevel,indexdelete -loglevel  ；名稱 >   | 設定記錄層級；可用層級為 info 、 debug （預設： info ）。 
|===


== indexdelete -match <filter>

使用 `-match <filter>` 參數 `indexdelete` 僅處理與篩選器相符的檔案和目錄的命令。

.語法
[source, cli]
----
xcp indexdelete -match <filter>
----
.顯示範例
[%collapsible]
====
[listing]
----
[root@client1 linux]# ./xcp indexdelete -match "fnm('S3_index12')"

== indexdelete -loglevel <name>

使用 `-loglevel <name>` 參數 `indexdelete` 用於設置日誌級別的命令；可用級別爲 info 和 debug 。預設層級為資訊。

.語法
[source, cli]
----
xcp indexdelete -loglevel <name> -match <filter>
----
.顯示範例
[%collapsible]
====
